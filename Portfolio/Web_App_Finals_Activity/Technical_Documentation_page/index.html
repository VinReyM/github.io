Introduction

HTML is a markup language used by the browser to mainpulate text, images, and other content, in order to display it in the required format.

HTML was created by Tim Bernes-Lee in 1991. The first ever version of HTML was HTML 1.0, but the first standard was HTML 2.0, published in 1995.

HTML Release Year:

  • HTML 1 (1993)
  • HTML 2 (1995)
  • HTML 3 (1997)
  • HTML 4 (1999)
  • HTML 4.01 (2012)
  • HTML 5 (2014)

HTML is a very important language as it is widely used in creating websites. Most of the websites are built using HTML5 (the latest version of HTML).

HTML mainly contains tags and attributes that determine the formatting and structure of a website.

What is HTML

HTML or HyperText Markup Language is the standard markup language for Web pages.

It defines the structure and content of web pages such as headings, paragraphs, images, links, lists, forms, tables. and buttons.

HTML Elements

HTML elements are the building blocks of web pages, representing different types of content or functionality. Each element is enclosed by a start tag and an end tag, with the content residing between them. HTML tags are not case-sensitive, but it is recommended to use lowercase for consistency. Some HTML elements, such as the tag, are empty elements and do not have an end tag

HTML elements can be nested, meaning that one element can contain another. For example, the (html) tag contains the (head) and (body) tags, forming a nested structure

The basic structure of an HTML page includes the doctype declaration, HTML, head, title, and body elements.

HTML Attributes

HTML attributes are special words used inside the opening tag of an HTML element to control the element's behavior or provide additional information about it.

They are a modifier of a HTML element type and can modify the default functionality of an element type or provide functionality to certain element types

There are several types of HTML attributes, including required attributes, optional attributes, standard attributes, and event attributes. Required attributes are needed by a particular element type for that element type to function correctly, while optional attributes are used to modify the default functionality of an element type. Standard attributes are supported by many element types, and event attributes are used to cause element types to specify scripts to be run under specific circumstances.
